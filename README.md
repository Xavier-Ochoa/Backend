SPRINT 2
En este Spring se desarrolló el registro de veterinarios: se creó el modelo, se validaron los datos en el controlador, se encriptó la contraseña, se generó un token y se envió un correo de confirmación.
Además, se configuró el envío de emails con Nodemailer, se agregó la ruta POST /api/registro y se integró Cloudinary para la gestión y almacenamiento de imágenes relacionadas con los veterinarios.
